# gosync

Sync files, fast.

# Installation

Clone the repo:

    get clone https://github.com/brettweavnet/gosync

Chagne in the gosync directory and run make:

    cd gosync
    make

# Setup

Set environment variables:

    AWS_SECRET_ACCESS_KEY=yyy
    AWS_ACCESS_KEY_ID=xxx

# Usage

    gosync source target
